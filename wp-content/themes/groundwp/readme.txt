Super fast, lightweight, AMP-ready theme for Bloggers and Marketers.

== Description ==

GroundWP is a super fast, lightweight, AMP-ready theme for Bloggers and Marketers built on the principles of:
* Accessibility
* [Lazy-loading of images](https://developers.google.com/web/fundamentals/performance/lazy-loading-guidance/images-and-video/)
* Mobile-first
* Progressive enhancement
* Progressive Web App enabled
* AMP-ready

=== AMP-ready ===
GroundWP is AMP-ready meaning it works hand-in-hand with the official AMP plugin and does not require further configuration to work with AMP features.
